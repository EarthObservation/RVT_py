RVT Python Package
==================

The RVT Python package (`rvt`) contains three modules:

* vis (:ref:`rvt.vis`) - visualization functions

* blend (:ref:`rvt.blend`) - blender (mixer)

* default (:ref:`rvt.default`) - default values
